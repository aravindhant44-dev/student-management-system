# 🎓 Student Management System

## 📌 Project Description

This is a Django-based web application to manage student records like adding, updating, deleting, and searching students.

## 🚀 Features

* ➕ Add Student
* ✏️ Update Student
* ❌ Delete Student
* 🔍 Search Student

## 🛠️ Technologies Used

* Python
* Django
* HTML, CSS

## ▶️ How to Run

```bash
python manage.py runserver
```
